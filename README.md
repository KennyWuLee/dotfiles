# dotfiles

## install

to get submodules
`git submodule update --init --recursive`

to create symlinks
`stow -t ~ vim`

for vim on windows
`mklink /D vimfiles Documents\GitHub\dotfiles\vim\.vim`

install oh-my-zsh from script first
