# dotfiles

## install

to get submodules
`git submodule update --init --recursive`

to create symlinks
`stow -t ~ vim`

for vim on windows
`mklink /D vimfiles Documents\GitHub\dotfiles\vim\.vim`

## zsh

for command not found support on arch:

- install 'pkgfile'
- `pkgfile --update`
